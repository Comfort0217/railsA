# README

## Dependencies
- ruby
- yarn
- sqlite

## Rubyバージョン 
- `.ruby-version`を参照してください

## 環境構築
- GitHubを操作して自分のリポジトリにこのリポジトリをforkした後自分のローカル環境にcloneしてください

``` 
$ bundle install
$ bin/rails db:setup
$ yarn
```

## 起動
```
$ bin/rails s
```


## テスト

``` 
$ bundle exec rspec
```

## 課題提出方法
- プルリクエストを作成し、カリキュラムからレビュー依頼を行ってください
